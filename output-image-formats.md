# تنسيقات الصور الناتجة

### TIFF (16 بت)

16 بت (رقم رقمي للبكسل 0 - 65535) تنسيق TIFF (.tif) الناتج.

### TIFF (32 بت، نسبة مئوية %)

32 بت (رقم بكسل عائم 0.0 - 1.0) إخراج بتنسيق TIFF (.tif). تمثل قيم البكسل نسب الانعكاس المئوية حيث 0.0 = 0٪ و 1.0 = 100٪.

### **JPG (8 بت)**

إخراج بتنسيق JPG (.jpg) 8 بت (رقم رقمي للبكسل 0 -255).

### **PNG (8 بت)**

إخراج بتنسيق PNG (.png) 8 بت (رقم بكسل رقمي 0 - 255). ضغط بدون فقدان مناسب للعرض على الويب والتصور.
